# AI Shipping Labs Personalized Plans

Personalized plan input collected from AI Shipping Labs community members. Each file is an internal working document used to prepare an individual learning plan.

All plan files follow the same rule:

- `Summary` and `Plan` come first and are the only sections intended to be shared with the member.
- Everything from `Internal Context` onward stays internal.

## File naming

Personal plan files in this folder use a `YYYYMMDD_` prefix based on the file's creation date (the `created` field in the frontmatter). The prefix makes it easy to find recent plans by sorting the directory listing. The order in the table below stays sorted by last updated, not creation - that is independent of the filename.

The prefix only applies to personal plan files in this folder. Index files (`_index.md`), the template (`_plan.md`), and shared notes (e.g. `how-to-use-plans.md`) keep their existing names.

The canonical structure lives at [_plan.md](_plan.md). Copy it to `YYYYMMDD_{person-name}.md` when preparing a new plan, using today's date as the prefix.
