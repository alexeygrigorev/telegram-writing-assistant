# Plan: Jakob Zischka

Internal working document. Share only the `Summary` and `Plan` sections with the member.

## Summary

- Current situation: in a career transition, motivated to become an AI engineer, but still building basic technical confidence and finding theory-heavy learning overwhelming.
- Goal for the next 6 weeks: move from "engaged AI observer" to "AI builder" by shipping one small working AI application.
- Main gap to close: Python and engineering foundations, plus the lack of a clear routine with fast, visible wins.
- Weekly time commitment: about 10 to 20 hours.
- Why this plan is the right next step: Jakob does not need more content discovery; he needs a simpler path, quick wins, and strong accountability while building something real.

## Plan

### Focus

- Main focus: strengthen Python and practical engineering confidence without getting buried in theory.
- Supporting focus: run the AI Hero course alongside the Python course for early AI-building wins.
- Supporting focus: choose one small personal project and grow it gradually from local utility to a simple AI app.

### Timeline

Week 1:

- Go through module 2 of Alexey's Python course as revision.
- Restart AI Hero from day 1.
- Install and use Claude Code for one small local task.
- Start keeping a note of daily annoyances that could become project ideas.

Week 2:

- Work through module 3 of the Python course.
- Continue AI Hero and aim to get a few days in by the end of the week.
- Use Alexey's brainstorming prompt and finish the week with 3 to 5 candidate project ideas.

Week 3:

- Commit to one project.
- Build the smallest useful local version first: simple data model plus CLI or similar non-AI workflow.
- Continue AI Hero and use module 4 of the Python course as needed.

Week 4:

- Add the first AI integration with a single API call.
- Keep the goal narrow: input -> API -> useful output.

Week 5:

- Finish the remaining AI Hero material.
- Apply at least one concrete thing learned there back into the project.

Week 6:

- Clean up the code with Claude Code's help.
- Write a short README.
- Demo the project in the community accountability channel and decide on the next 6-week arc.
